Explanation

A compact calendar, local cache, and APOD text stay together in one quiet desktop shell.

What it does

Built for the tiny daily ritual of changing your wallpaper.

01

Calendar-first browsing

See local images, available APOD image days, video days, and unchecked days without opening a browser.

02

Preview before apply

Load the preview and explanation text first, then download or apply only when the date is worth it.

03

Automatic daily checks

Run quietly from the system tray and apply the latest available image when auto-check is enabled.

04

Wallpaper fit modes

Use Smart, Fill, Fit, Stretch, Center, Tile, or Span. Smart mode avoids ugly crops where possible.

Smart mode

Different space photos, one desktop that still feels intentional.

APOD images come in every shape: square, ultra-wide, vertical, ultra-vertical, tiny, enormous. Smart mode prepares the selected image for your screen resolution and tries to preserve the subject instead of blindly cropping the best parts away.

→

Square image

Centered composition with a desktop-friendly background instead of harsh edge cropping.

→

Vertical image

Portrait APODs can stay readable without being stretched into a distorted mess.

→

Ultra-wide image

Very wide frames are handled carefully so the left and right edges are not casually thrown away.

→

Ultra-vertical image

Extremely tall images can be presented without pretending they were born as desktop wallpapers.

Privacy

Local-first by design.

APOD Wallpaper stores settings and optional NASA API key data locally on your device. It contacts NASA APOD services to fetch metadata, images, and APOD pages.
