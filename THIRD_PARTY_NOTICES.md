# Third-Party Notices

As of April 29, 2026, this repository does not use any third-party NuGet packages.

The solution references only Microsoft .NET Framework and Windows desktop assemblies that ship with the target platform or its developer tooling, including:

- `Microsoft.CSharp`
- `System`
- `System.Core`
- `System.Data`
- `System.Data.DataSetExtensions`
- `System.Deployment`
- `System.Drawing`
- `System.Net.Http`
- `System.Runtime.Serialization`
- `System.Security`
- `System.Windows.Forms`
- `System.Xml`

These framework assemblies are not redistributed here as third-party source dependencies.
